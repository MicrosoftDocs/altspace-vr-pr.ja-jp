# <a name="creating-and-linking-your-altspacevr-account"></a>AltspaceVR アカウントの作成とリンク

## <a name="disclaimer-regarding-account-linking"></a>アカウントリンクに関する免責事項

AltspaceVR アカウントを作成してリンクするさまざまな方法を次に示します。

アカウントを作成または更新する場合は、AltspaceVR の "パスワード" を必要とする機能がまだ存在します。

MSA でアカウントを作成し、AltspaceVR パスワードを持ってない場合は、 にサインインし、プロファイルに移動し、[サインアウトして新しいパスワードを取得する] をクリック https://www.altvr.com/ します。


## <a name="how-to-create-an-account-in-vr"></a>VR でアカウントを作成する方法

Microsoft ログインを使用したアカウントの作成:

1. まず、VR アプリ ストアまたはライブラリから AltspaceVR のライブ ビルドをダウンロードします。 <br>*アプリケーションが既にインストールされている場合は、最新の更新プログラムがインストールされていることを確認します。*
2. アプリケーションを起動し、左側の [サインイン] ボタンに移動します。
3. [サインイン] ボタンを選択し、[サインイン] ボタンMicrosoft アカウントでサインイン選択します。
4. 別の画面が表示され、一意のコードを使用してデバイスをアクティブ化するように求めるメッセージが表示されます。このコードをコピーする必要があります。
6. コンピューターのブラウザー画面に切り替えて に移動します https://www.microsoft.com/devicelogin
7. 一意のコードを入力し、[次へ] を選択します。 *コードでは大文字と小文字は区別されません。*
8. 画面の指示に従ってデバイスをアクティブ化します。
9. アカウントリンク プロセスが完了すると、表示名とユーザー名の入力を求めるプロンプトが表示されます。
10. [登録] を選択します。

## <a name="how-to-create-an-account-in-2d-mode"></a>2D モードでアカウントを作成する方法

Microsoft ログインを使用したアカウントの作成:

1. まず、 [お好みのアプリ ストアまたはライブラリから AltspaceVR](https://altvr.com/getaltspacevr) のライブ ビルドをダウンロードします。 <br>*アプリケーションが既にインストールされている場合は、最新の更新プログラムがインストールされていることを確認します。*
2. アプリケーションを起動します。
3. [サインイン] ボタンを選択し、[サインイン] ボタンMicrosoft アカウントでサインイン選択します。
3. 別のウィンドウが開き、使用する Microsoft アカウントを選択するように求めるメッセージが表示されます。 個人の Microsoft アカウントを選択します。仕事用または学校アカウントに関連付けられているアカウントは選択しません。
4. アカウントリンク プロセスが完了すると、表示名とユーザー名の入力を求めるプロンプトが表示されます。
5. [登録] を選択します。

## <a name="linking-your-microsoft-account"></a>Microsoft アカウントのリンク

既存の AltspaceVR アカウントは、AltspaceVR アプリケーション内からリンクできます。

PC 上の 2D モードとのリンク:

1. 最新の更新プログラムがインストールされていることを確認し、AltspaceVR を起動します。
2. AltspaceVR を起動し、クラシック AltspaceVR アカウントでサインインしている必要があります。 AltspaceVR の起動時にサインインするように求めるメッセージが表示されたら、[サインイン] ボタンを選択し、[Or sign in with an AltspaceVR account]/(AltspaceVR アカウントでサインインする)リンクを選択します。
2. [タイトル画面] で、右側の [アカウント] タブを選択します。 アカウントがリンクされていない場合、アカウントの状態は "クラシック Altspace アカウント" と読み取る必要があります
3. [Microsoft アカウントのリンク] を選択します。 正しいユーザー名が表示されていることを確認し、[リンク] ボタンを選択します。
4. 別のウィンドウが開き、リンクする Microsoft アカウントを選択するように求めるメッセージが表示されます。 個人の Microsoft アカウントを選択します。仕事用または学校アカウントに関連付けられているアカウントは選択しません。
5. リンクの手順に従います。 アカウントリンク プロセスが完了すると、メニューの [アカウント] セクションでアカウントの資格情報が更新されたのが表示されます。 状態は "Microsoft Home Account (MSA)" と表示されます

ヘッドセットでの VR モードとのリンク:

1. 最新の更新プログラムがインストールされていることを確認し、AltspaceVR を起動します。
2. AltspaceVR を起動し、クラシック AltspaceVR アカウントでサインインしている必要があります。 AltspaceVR の起動時にサインインするように求めるメッセージが表示されたら、[サインイン] ボタンを選択し、[Or sign in with an AltspaceVR account]/(AltspaceVR アカウントでサインインする)リンクを選択します。
2. [タイトル画面] で、右側の [アカウント] タブを選択します。 アカウントがリンクされていない場合、アカウントの状態は "クラシック Altspace アカウント" と読み取る必要があります
4. [Microsoft アカウントのリンク] を選択します。 正しいユーザー名が表示されていることを確認し、[リンク] ボタンを選択します。
4. 別の画面が表示され、一意のコードを使用してデバイスをアクティブ化するように求めるメッセージが表示されます。このコードをコピーする必要があります。
5. コンピューターのブラウザー画面に切り替えて に移動します https://www.microsoft.com/devicelogin
6. 一意のコードを入力し、[次へ] を選択します。 *注: コードでは、大文字と小文字は区別されません。*
7. 画面の指示に従ってデバイスをアクティブ化します。
8. AltspaceVR に戻った後。 メニューの [アカウント] セクションで、アカウントの資格情報が更新されているのが表示されます。 状態は "Microsoft Home Account (MSA)" と表示されます。

トラブルシューティングのヒント:

* アカウントリンク プロセス中に問題が発生した場合は、アカウントへの接続を再試行する前に AltspaceVR を終了して閉じてください。
* VR モードで問題が発生している場合は、ヘッドセットを取り外して、2D PC モード (またはそれ以外の方法) に切り替えてみください。
* ユーザー名とパスワードで再接続できない場合Microsoft アカウント、ユーザー名とパスワードを使用して AltspaceVR に戻る可能性があります。 [AltspaceVR でサインイン] ボタンを選択します。
* VR を使用している場合は、ダイアログを開くメッセージが表示されたら、AltspaceVR で使用する予定の Microsoft 資格情報を既に使用している PC で実行してみてください。
* [メイン] メニューの [設定] に移動して問題が発生した場合は、ログ>**サポート>送信> アップロードします**。

## <a name="account-registration-faq"></a>アカウント登録に関する FAQ

**他のユーザーに表示される名前は何ですか?**

[表示名] は、他のユーザーがアバターの上に表示する名前です。 [ユーザー名] は、フレンドとして追加したユーザーのメニューの [フレンド] タブに表示されます。 [表示名] と [ユーザー名] の両方が、アバターの名前タグに表示されます。

**表示名は一意である必要がありますか?**
 
いいえ。ユーザー名は、他のユーザーと同じ表示名を共有できます。 表示名を "Dave" または "Maria" として書き込む場合は、 [表示名] には、通常はユーザー名を使用できないスペースを含めすることもできます。

**パスワードを覚えていないので、リセットするにはどうすれば良いですか?**

- アプリを使用している場合Microsoft アカウント、ログイン プロセスを開始した後、[パスワードのリセット] ボタンを使用して Microsoft のパスワード リセット フロー Microsoft アカウントでサインインできます。 
- 従来の AltspaceVR アカウントを使用している場合は、[パスワードを忘れた場合] リンクを使用してパスワードをリセットするか、ここでパスワードをリセット [できます](https://account.altvr.com/users/password/new)。

**ユーザー名または表示名を変更する方法**

[ユーザー名 **]** または [表示名] を変更するには、altvr.com にサインインし、右上の [ユーザー プロファイル] 、 [編集] のプロファイル アイコンを選択し、ユーザー名または表示名を更新します。 

*注: ユーザー名を変更できるのは 1 回のみです。その後は、いつでもサポート チケットを送信する必要があります。*

アカウントに関連付けられているユーザー名を変更する場合は、 [チケット を送信してください](https://help.altvr.com/hc/requests/new)。 アカウントに関連付けられている電子メール、現在の名前、および今後使用するユーザー名を必ず含める必要があります。

**メニューが空白でサインアップできない場合、何ができますか?**

"You" メニューが表示されない場合は、AltspaceVR クライアントを再起動すると、サインインできるよう問題が解決されます。 VR ヘッドセットと 2D モニターの両方が同じ GPU に接続されていることを確認します。 問題が解決 [しない場合は、](https://help.altvr.com/hc/requests/new) ヘルプ デスクにお問い合わせください。

## <a name="microsoft-account-integration-faq"></a>Microsoft アカウント統合に関する FAQ

**これにより、AltspaceVR のユーザー名が変更されますか、または自分の詳細が名前タグに表示される方法は変わりますか?**

AltspaceVR アカウントを Microsoft アカウントにリンクしても、公開されているユーザー名や名は変更されません。 アカウントのログイン資格情報だけが変更されます (これはユーザーに対してプライベートです)。AltspaceVR では、名前、ユーザー名、およびプロファイル画像を他のユーザーに表示する方法を完全に制御できます。

**引き続きアカウントを切り替えできますか?このオプションは、このオプションの設定。**

はい、少し再設定しました。 メイン メニューの [アカウントの切り替え&アカウント **を管理>アカウント>できます**

**Microsoft アカウントを既に持っていますか?**

既に Microsoft アカウントを持っている可能性があります。 Xbox でMinecraftゲームをプレイしたり、Office 365 にサブスクライブしたり、他の Microsoft 製品やサービスを定期的に使用したりしていますか? その場合は、Microsoft ログインが既に存在している可能性があります。 電子メールが既存の MSA に既に関連付けられているかについては、こちらを参照 [してください](https://login.live.com/login.srf?wa=wsignin1.0&rpsnv=13&ct=1610764342&rver=7.0.6738.0&wp=MBI_SSL&wreply=https:%2F%2Faccount.microsoft.com%2Fauth%2Fcomplete-signin%3Fru%3Dhttps%253A%252F%252Faccount.microsoft.com%252F%253Frefp%253Dsignedout-index&lc=1033&id=292666&lw=1&fl=easi2)。

**これは、新しい電子メール アドレスを作成する必要があるという意味Outlook。**

不正解です。 Microsoft アカウントは、Microsoft でホストされていない場合でも、現在の電子メール アドレスで設定できます。 (例: G-Mail、Yahoo など)。 クラシック AltspaceVR アカウントと同じ電子メールを共有する Microsoft 資格情報でサインインすると、サインイン後にアカウントをリンクするように求めるメッセージが表示されます。 プロンプトに従ってクラシック アカウントにサインインすると、Altspace アカウントと Microsoft アカウントが自動的にリンクされます。

**コストはかかりますか?**

いいえ！ Microsoft アカウントの作成は無料です。 AltspaceVR の使用も無料です。 Microsoft アカウントは他の製品やサービスにリンクされています。そのため、サブスクリプション (例: Office 365) や特殊な商品 (例: Minecraft Marketplace) など、AltspaceVR の外部で購入することができます。

**アカウントのリンクで問題が発生した場合は、どうしますか?**

行き詰まっている場合は、いつでもヘルプ デスク にお問い [合わせください](https://help.altvr.com/hc/requests/new)。

**複数の AltspaceVR アカウントがある場合は、どうしますか?**

Microsoft アカウントにリンクできる AltspsaceVR アカウントは 1 つのみです。 接続する AltspaceVR アカウントごとに、個別の Microsoft アカウントが必要です。

**ログインの詳細を忘れた場合は、どうしますか?**

Microsoft アカウントに接続した後に AltspaceVR へのログインに問題がある場合は、ヘルプ デスク までお問 [い合わせください](https://help.altvr.com/hc/requests/new)。 Microsoft アカウントへのログインに関する問題[](https://support.microsoft.com/account-billing/when-you-can-t-sign-in-to-your-microsoft-account-475c9b5c-8c25-49f1-9c2d-c64b7072e735)については、[](https://support.microsoft.com/account-billing/how-to-help-keep-your-microsoft-account-safe-and-secure-628538c2-7006-33bb-5ef4-c917657362b9)トラブルシューティングのヒントとリソースの一覧については、こちらを参照してください。

**Microsoft アカウントに接続する必要がありますか?**

心配はいりません。更新を行う時間があります。 Microsoft アカウント統合を段階的に展開します。 ただし、時間の長い間、現在取り組む主要な新機能の一部は、AltspaceVR アカウントを自分の Microsoft アカウントにリンクしたユーザーだけが利用できます。 その理由は、これらの機能 (チケット付きイベントなど) には、強力で堅牢なセキュリティ対策が必要だからです。 詳細については、ブログを [参照してください](https://altvr.com/microsoft-account-integration)。

**プライバシーとデータについて**

サービス利用規約やプライバシー ポリシーに[変更](../community/terms-of-service.md)[はありません](https://privacy.microsoft.com/privacystatement)。 Microsoft と AltspaceVR は、お客様のプライバシーの保護に引き続きコミットしています。 また、データと情報に関しては、引き続き完全に制御できます。 プライバシーに関する Microsoft の主要な原則の詳細については、こちらを参照 [してください](https://privacy.microsoft.com)。 Microsoft が Microsoft アカウントに関連するユーザーに関して持っている個人データに関する詳細情報は、こちらから要求 [することもできます](https://www.microsoft.com/concern/privacyrequest-msa)。 最後に、アクティブな AltspaceVR アカウントがある場合は、データのコピーを要求し、いつでもデータの削除を要求できます。
