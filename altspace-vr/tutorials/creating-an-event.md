これは、Altでイベントを作成するためのステップバイステップガイドです。 どのように動作するかを確認するには、Altwork Evr でいくつかのイベントに参加することを強くお勧めします。 すべてのイベントの一覧については、 [Altの Evr イベントカレンダー](https://account.altvr.com/events) を確認してください。

この記事では、次の内容について説明します。

* Altて Evr イベントの用語
* イベントとイベント領域または世界の作成
* スライドプレゼンテーションにマルチメディアコンソールを使用する

## <a name="understanding-events"></a>イベントについて

Altspace Evr を使用すると、事前に作成されたテンプレート領域を使用して特定のニーズに合わせてカスタマイズしたり、世界をゼロから作成したりすることができます。 この記事では、事前に作成したテンプレートを使用してイベントを作成する方法について説明します。 次のセクションでは、 [イメージを使用したイベントのブランド](#branding-your-event-with-images) 化と、 [ワールドエディターとキットを使用した3D オブジェクトの追加](#adding-3d-objects-using-world-editor-and-kits) について、さらにカスタマイズするためのヒントを紹介します。 カスタム世界をゼロから構築する方法の詳細については、ミートアップの世界ビルのツアーに関する記事を参照[してください。](../world-building/world-editor-getting-started.md)

Altspace Evr には、イベント用の領域を作成するための2つの方法が用意されています。

* **1 回限りの使用:** イベントを作成し、テンプレートの世界を選択します。
* **繰り返し使用:** ワールド空間を作成し、イベントにインポートします。

イベントを作成するプロセスは、どちらの場合も同じです。ただし、 [世界を作成](../world-building/managing-worlds.md) し、繰り返し使用するイベント領域としてインポートするという1つの例外があります。 次の点にも注意してください。

</br>

| 項目 | 定義 |
|---|---|
| コピーされたワールド | 繰り返しイベントにワールド空間を使用すると、永続的なカスタマイズが可能になります。 署名、画像、ポイントの生成、およびその他のカスタマイズは、イベント領域を毎回カスタマイズするのではなく、イベントからイベントに保持されます。 |
| イベント世界のカスタマイズ | イベントが作成されると、テンプレートまたはインポートされた世界がコピーされ、そのイベントにロックされます。 イベントの世界には変更を加えることができ、元の世界には影響を与えません。また、もう1つの方法もあります。 イベント領域をより楽しく、イベントに適したものにするために、休日、画像、またはその他の装飾的な1回限りの使用環境のエディター要素を追加することを検討してください。 |

元の世界に加えられた変更は、イベント領域が更新されない限り、イベントの世界には反映されません。
1. イベント web ページの [ **WORLD の再インポート** ] ボタンを使用します。
2. 同期が完了するまでに2-3 分を許可します。 
3. イベント領域が変更されていない場合は、[設定] > [中 **程度] > [領域のリセット** ] にアクセスして、イベント領域をリセットします。 世界のビルダーとしてスペースをリセットします。

要素が適切に読み込まれていないなど、技術的な問題が発生した場合は、[ **Altspace** evr] メニューにアクセスし、[設定] を選択して [> >] を選択し、イベント領域をリセットし、新しい変更を確認します。 2D の Windows PC ユーザーは、ショートカットキーを使用できます。 **CTRL + ALT + r** を使用すると、領域をすばやくリセットできます。

## <a name="creating-your-event-and-event-space-or-world"></a>イベントとイベント領域または世界の作成

ここでは、1回限りのイベントまたは繰り返し発生するイベントのイベントを作成する手順について説明します。 テンプレートを選択するか、イベントの世界をインポートします。 

> [!NOTE]
> Altworkplace Evr イベントの web ページでは、緑の疑問符のボタンを使用して、フォームのあらゆる側面に関する指示を提供しています。 特定の手順については、カーソルを上に重ねてください。

[Alt] web サイトの [イベント [> マイイベント](https://account.altvr.com/events/my) ] web ページで、[ **イベントのスケジュール設定** ] を選択するか、または [ [altworkplace evr] web ページで [イベントの作成](https://account.altvr.com/events/new)] に直接進みます。

1. **イベントのタイトル:** イベントの名前を入力します。 これは、Altworkplace Evr の web サイトと世界中のインターフェイスのさまざまなカレンダービューで表示するために、明確で簡潔なものにしてください。 文字の間にスペースを含めて、タイトルを23文字以内にしてください。
2. **説明:** イベントの説明を入力します。
    * 説明には10文字以上の文字を含める必要があります。そうしないとイベントは作成されません。
    * 段落間に空のスペースを追加します。
    * イベントの Facebook、Discord、web サイト、またはその他のリソースへのリンクを追加するには、次の形式を使用します (この Markdown は、web サイトのイベントのプロモーションページでのみ機能します。これは、Alt Evr メニューで正しくレンダリングされません)。 `[Event Name](http://example.com/)`

3. **開始日/終了日:** 開始時刻を設定し、終了時刻が開始時刻より後であることを確認します。
4. **カテゴリ:** イベントの種類に最も適したカテゴリを選択します。 
5. イベントを **プライベート** または **パブリック** に設定します。
    * 可視性: パブリックイベントは、Altare Evr web サイトの [ [すべて] タブ](https://account.altvr.com/events/all) に表示され、パブリックに表示されます。
    * プライベートイベントは、Altspace Evr イベントのカレンダーには表示されず、イベントの場所を入力するためにイベント URL を必要とします。
    * "メインイベントとして追加" する場合は、イベントを [パブリック] に設定する必要があります。
6. **テンプレートを選択してください:** Web ページの右側には、Altworkplace Evr で使用できるテンプレートのサムネイル画像の一覧が表示されます。 ゲームルーム、オフィス、ミーティングスペース、プレゼンテーションスペース、楽しい meetup スペースがあります。 興味深いものを選択します。 気に入らない場合は、新しいテンプレートを使用して新しいイベントを作成するだけです。 独自のカスタムイベントワールドを作成する場合は、既定のテンプレートとして **クールスカイ** を選択し、次の手順に従って、世界をインポートします。
7. **詳細オプション** の選択

### <a name="advanced-options"></a>[詳細オプション]

1. **昇格:** 各イベントには2つのブランド化イメージが必要です (これらのイメージは Altworkplace Evr web サイトに表示され、Altworkplace Evr 内のイベントには表示されません)。
    * **タイル:** タイルイメージは、1920 x 1080 ピクセルである必要があります。 この画像はさまざまなサムネイルサイズにスケーリングされ、他のイベントと共に Altのイベントカレンダーに表示されます。 画像がはっきりしていて、それほど多くのテキストが含まれていないことを確認します。 自分が所有していない著作権のあるイメージやロゴは使用しないでください。
    * **バナー:** バナー画像は1920x576 である必要があります。 必要に応じて自動的にサイズが変更され、イベント情報または web ページに表示されます。 半透明なオーバーレイは、イメージの下位25% をカバーします。 その領域にテキストを配置しないようにします。
    * もう1つの方法として、 **タイルイメージをバックグラウンドバナーイメージにコピー** する方法もあります。これにより、タイル画像がバナー画像としても使用されます。 :) を試してみてください。 正しく表示されない場合は、新しいバナーイメージを作成できます。
2. **VR の場合:** このセクションには、Altworkplace Evr アプリ内のイベント中の仮想エクスペリエンスに適用される機能が含まれています。
    * **既定のコンテキストロール:** 緑色の疑問符は、 *イベントのすべての対象ユーザーメンバー* が使用できる特定のロールを一覧表示します。 最も一般的なのは *megaphone_only* です。 これを追加すると、[ホストツール] ボタンの下にある [音声の増幅] オプションがすべての対象ユーザーのメンバーに表示されます。これにより、大きなスペースになっている場合にイベントで通知される可能性があります。 イベントにフライトが必要な場合は、 *パイロット* を追加します。 両方を追加するには、次の形式を指定します。 *megaphone_only、パイロット* です。 各ユーザーは、[設定]、[入力/実行] の順に移動して、Altを使用してフライモードを有効にする必要があります。
    * **手順:** この [手順](../world-building/adding-welcome-messages.md) は、スペースに到着したときにウェルカムイメージを生成するテキストを対象としています。 ユーザーは [OK] を選択して、ビューから削除する必要があります。 多くの場合、この方法は、"音声に招待されるまでミュート状態を維持する" や "XYZ へようこそ" といった対象ユーザーの指示を提供するために使用されます。 これは必須ではないので、慎重に使用してください。 Markdown を使用して、色とフォントのサイズを追加することもできます。詳細については、以下を参照してください。 [http://digitalnativestudios.com/textmeshpro/docs/rich-text](http://digitalnativestudios.com/textmeshpro/docs/rich-text)
3. **詳細設定:** 次に、イベントの高度な機能について簡単に説明します (これらの機能の一部は、イベントが作成されてイベントが編集されるまで表示されません)。
    * **管理者:** 管理者は、イベントの管理を支援するために信頼できるユーザーです。 これらは、cohosts または backup ホストである場合があります。 管理者リストにユーザー名を追加すると、次のことが可能になります。
        * イベントのタイトル、説明、およびその他の機能を変更します。
        * モデレーター、プレゼンター、およびその他のロールのコンテキストロールを追加および削除します。
        * イベントを削除します。
    * **グループ:** ドロップダウンメニューを使用して、プライベート [グループ](group-features.md) から選択します。 アカウントがグループに作成/追加されている場合にのみ表示されます。
    * **タグライン:** この簡潔な文は、イベント web ページのタイトルの下に表示されます。
    * **世界からのインポート:** 作成されたワールドイベント領域を繰り返し使用するために使用するには、これが、その世界のレイアウトとカスタマイズをイベントにインポートするために使用するフォームのセクションです。 次の考慮事項に注意してください。

    **自分の世界をインポートします。** 世界をインポートするには、次のように作成して所有します。
    * ドロップダウン矢印を選択すると、作成した世界の一覧が表示されます。
    * 世界を選択します。
    * イベントフォームの残りの部分を完了したら、少なくとも2分待ってから、データベース情報が更新され、世界が生成されるようにします。
    
    **他のユーザーの世界を使用する:**
    * 世界の所有者に連絡して、個人の世界で **共有と共有** を使用するよう依頼してください。

5. **コンテキストロール:** ここでは、イベントホスト、プレゼンター、モデレーター、およびイベント内のその他のロールのスーパーパワーまたは [コンテキストロール](../world-building/granting-roles.md) を制御します。 それらを追加するには、[Altlt Evr username] \ (ユーザー名 \) に「」と入力し、ドロップダウンメニューでロールを割り当てます。 削除するには、イベント web ページを削除して保存するチェックボックスをオンにします。 注: プレゼンターのモデレーション機能を付与する場合は、それらをプレゼンターおよびモデレーターとして追加する必要があります。 現在、次のコンテキストの役割を使用できます。

    * **音楽の実行者:** コンサートおよび音楽イベントに関連付けられた機能を追加します。
6. **リストされたユーザーをブロックする:** ユーザーによるアクセスをブロックする場合、またはユーザーが以前にモデレーターまたはプレゼンターによってイベントから削除され、イベントが重複している場合は、リストされているユーザーの名前が一覧表示されます。 イベントホストまたは管理者は、表示されているブロックユーザーをいつでも追加または削除できます。

フォームが完成し、トリプルチェックされたら、[ **イベントの作成**] を選択します。

## <a name="event-page-actions"></a>イベントページのアクション

イベント web ページには、次のアクションとオプションがあります。

1. イベント設定を変更するには、[ **編集** ] を選択します。
2. イベントを終了するには、[イベントの **終了** ] を選択します (イベントが早期に発生している場合など)。
3. [ **ドラフトに設定** ] を選択して、イベントをアクティブではなくドラフトに設定します。
4. インポートした世界に更新を行った場合は、イベントに表示されるように変更するには、[ **世界の再インポート** ] を選択する必要があります。イベントの領域をリセットすることは忘れないでください:)
5. [ **重複イベント** ] を選択して、将来のイベントに対してイベントを複製します。 この重複は [マイイベント]/[My Draft] イベントに表示されます。新しい日付/時刻を更新し、そこからアクティブ化する必要があります。
6. [ **メインイベントとして追加** ] を選択して、イベントを予定表の一覧に設定します。
7. イベントを完全に削除する場合は、[ **イベントの削除** ] を選択します (これは復旧できません)。

準備ができたら、Altspace Evr の [ **>** のイベント] メニューにアクセスして、領域をリセットするためのイベント領域を入力するか、変更を検査するか、またはカスタマイズを続行します。

> これは、このイベントのみを対象とする、コピーとイベント領域のカスタマイズです。 イベントの領域をカスタマイズするために多くの時間を費やしている場合は、 **繰り返し使用するイベント** のために世界を作成する方がよい場合があります。

## <a name="branding-your-event-with-images"></a>画像を使用してイベントをブランド化する

イベント空間または会社のロゴとブランド、または自分の写真、チーム、その他の視覚的なコンテンツに画像を追加するのは、Altspace Evr で簡単に行うことができます。

イメージの要件は次のとおりです。
* JPG または PNG ファイルが受け入れられる
* イメージの最大サイズ: 1920 x 1080
* 解決策:72 DPI
* ファイルサイズ: ファイルあたり 250 KB 未満

このプロセスでは、イメージを [altspace Evr の写真](https://account.altvr.com/photos)にアップロードし、世界中のエディターの写真を使用して、イベントのワールド空間にインポートして配置します。

1. Altworkplace Evr web サイトの [写真](https://account.altvr.com/photos) にアクセスします。
2. **[アップロード]** を選択します。
3. [ **参照** ] を選択して、コンピューター上でダイアログボックスを開き、最適化されたイメージを探して選択します。
4. **[Open]** を選択します。
5. [ **写真の作成**] を選択します。
6. すべてのイメージに対してこの手順を繰り返します。

Altplace Evr およびイベントの世界では、イメージを配置する場所の近くに配置します。 (これを **繰り返し使用するイベント** 用に作成している場合は、イベント領域ではなく、世界中のユーザーであることを確認してください)。

1. ユーザーインターフェイスの右下隅にある [**ワールドエディター]/[エディターパネル**] を選択します。
2. 下部にある [ **自分** のもの] タブをクリックします。
3. **写真** を選択します。
4. 写真はアップロードの時間順に保存されるため、アップロードされた最新のイメージは一覧の最初のイメージになります。 これを選択して、世界に追加します。
5. デバイスカーソルを使用して、イメージを取得し、目的の場所に配置します。
    * コントローラーを使用してサイズを変更することができます (通常は、VR のタッチパッド全体で thumb を水平にスライドさせるか、2D モードでマウスを使用します)。
    * 画像の位置と回転を微調整するには、エディターでイメージの **歯車** 記号を選択し、回転と位置を適切に設定します。
    * また、スケール機能を使用してイメージのサイズを変更することもできます。

> [!NOTE]
> 編集モードがオン (オレンジ色のテキスト) になると、自動的にフライトが可能になり、イベントの世界でのイメージの配置がより簡単になります。

6. イメージの配置ごとにこの手順を繰り返します。
7. イメージが適切に設定されている場合は、個別にロックしていない場合は [ **すべてロック** ] を選択し、編集モードをオフにして、エディターを閉じます。
8. イベントあたり最大で5枚の写真を使用します。

## <a name="adding-3d-objects-using-world-editor-and-kits"></a>ワールドエディターとキットを使用した3D オブジェクトの追加

Altworkplace Evr を使用すると、ワールドエディターで3D オブジェクトをワールドにインポートできます。 現在、使用可能なさまざまなエディターキットが用意されていますが、その方法もあります。 宇宙船、rockets、アニメーション化された動物、sculptures、木、植物、およびさまざまなオブジェクトがあります。 または、独自のキットをパブリックコレクションに追加することも、個人の使用を目的として使用することもできます。または、対話型オブジェクトをヘルメット、swords、主翼、衣料、帽子などの対話型オブジェクトとして使用する [ことを検討](../world-building/using-mixed-reality-extensions.md)してください。 オブジェクトの一覧が間もなく拡張されることが予想されます。

ここでは、既にキットに含まれている項目の使用について説明します。 3D オブジェクトを Altに追加してインポートする方法の詳細については、「 [glTF モデルを自分の世界にインポートする方法」](../world-building/importing-models.md)を参照してください。 また、 [「独自のキットをアップロード操作方法ですか](../world-building/uploading-custom-kits.md)。

世界中のビルドキットからイベント世界に既存の3D オブジェクトを追加するには、[ワールドエディター]/[エディター] パネルを開いて編集モードを有効にします。

1. インポートするオブジェクトのキットを探索して選択します。
2. カーソルを使用してキットオブジェクトを選択し、その位置に位置付けます。 ワールドエディターのオブジェクトの歯車アイコンを使用して、手動の番号入力の位置、回転、およびスケールのオプションを使用します。
3. オブジェクトが右側の位置に配置されたら、ロックアイコンをクリックしてロックします。
4. 必要に応じて、別のオブジェクトを追加します。 3D オブジェクトを追加する際には慎重に行う必要があります。5つの異なる個別のキットの項目を選択したままにしておくと、Oculus クエストユーザーサポートが向上します。 世界では、200を超えるオブジェクトを使用しないでください。
5. 完了したら、すべての項目をロックし、編集モードをオフにして、ワールドエディターを閉じます。 移動します。 いくつかのテスト担当者を招待して、アクセスして確認し、優れたイベントの準備をしましょう。

ワールドエディターキットの詳細と、次のように3D オブジェクトを Altworkplace Evr に追加する方法については、以下を参照してください。

* [早期アクセスプログラムとは何ですか。](../world-building/early-access.md)
* [GlTF モデルをワールドにインポートする方法](../world-building/importing-models.md)
