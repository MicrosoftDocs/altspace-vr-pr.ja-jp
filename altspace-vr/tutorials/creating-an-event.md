# <a name="creating-an-event"></a>イベントの作成

これは、AltspaceVR でイベントを作成するためのステップ バイ ステップ ガイドです。 AltspaceVR のいくつかのイベントに参加して、その動作を確認することを強くお勧めします。 すべてのイベント [の一覧については、AltspaceVR イベント](https://account.altvr.com/events) カレンダーを確認してください。

この記事では、次の内容について説明します。

* AltspaceVR イベントの用語
* イベントとイベント空間またはワールドの作成
* スライド プレゼンテーションにマルチメディア コンソールを使用する
* 画像を使用したイベントのブランド化
* ワールド エディター/キットを使用した 3D オブジェクトの追加
* マイ イベントを記録またはLive Streamする方法

## <a name="understanding-events"></a>イベントについて

AltspaceVR を使用すると、事前に作成されたテンプレート空間を使用して、特定のニーズに合わせて使用またはカスタマイズしたり、ゼロから世界を作成したりできる世界を簡単に作成できます。 この記事では、事前に作成されたテンプレートを使用してイベントを作成する方法について説明します。 以下の「画像を [使用した](#branding-your-event-with-images) イベントのブランド化」および「ワールド エディターとキットを使用した [3D オブジェクト](#adding-3d-objects-using-world-editor-and-kits) の追加」のセクションには、さらにカスタマイズするためのヒントが含まれています。 カスタム ワールドをゼロから構築する方法については、AltspaceVR の World Building Tour の会議に参加し、World Building のサポート ドキュメント [を参照してください](../world-building/world-editor-getting-started.md)。

AltspaceVR には、イベント用のスペースを作成する 2 つの方法が用意されています。

* **1 回の使用:** イベントを作成し、テンプレートワールドを選択します。
* **繰り返し使用:** ワールド空間を作成し、 イベントにインポートします。

イベントを作成するプロセスはどちらも同じですが、1 つの例外があります[](../world-building/managing-worlds.md)。ワールドの作成と、それを繰り返し使用イベント空間としてインポートします。 また、次の情報も必要です。

</br>

| 期間 | 定義 |
|---|---|
| コピーされたワールド | 繰り返しのイベントにワールド空間を使用すると、永続的なカスタマイズが可能になります。 記号、画像、生成ポイント、その他のカスタマイズは、イベント空間をカスタマイズするのではなく、イベントからイベントに維持されます。 |
| Event World のカスタマイズ | イベントが作成されると、テンプレートまたはインポートされたワールドがコピーされ、そのイベントにロックされます。 イベントの世界に変更を加え、元の世界に影響を与え、それ以外の方法で影響を与える可能性があります。 休日の装飾、画像、または他の装飾的な 1 回きり使用のワールド エディター要素を追加して、イベント空間をより楽しく、イベントに適したものにしてください。 |

イベント空間が更新されていない限り、元のワールドに対して行われた変更はイベントワールドに含め "されません"。
1. イベント Web **ページの [RE-IMPORT WORLD]** ボタンを使用します。
2. 同期が完了するには、2 分から 3 分かかります。 
3. イベント領域が変更されなかった場合は、 [設定] > [モデレート] > **に** 移動して、イベント領域をリセットします。 ワールド ビルダーとして領域を多くリセットします。

要素が正しく読み込まれないなど、技術的な問題がある場合は、AltspaceVR メニューに移動し、[設定] > [モデレート] **>** [領域のリセット] を選択してイベント領域をリセットし、新しい変更を確認します。 Windows PC 2D のユーザーは、AltspaceVR で Ctrl **+ Alt + R** キーを押すと、スペースをすばやくリセットできます。

## <a name="creating-your-event-and-event-space-or-world"></a>イベントとイベント空間またはワールドの作成

1 回または繰り返しのイベントを作成するための詳細な手順を次に示します。 テンプレートを選択するか、イベントのワールドをインポートします。 

> [!NOTE]
> AltspaceVR イベント Web ページでは、緑色の疑問符ボタンを使用して、フォームのあらゆる側面に関する手順を提供します。 特定の手順のために、カーソルをロールオーバーします。

AltspaceVR [>の Web](https://account.altvr.com/events/my)ページの [イベント] ページで、[イベントのスケジュール設定] を選択するか[、[AltspaceVR](https://account.altvr.com/events/new)でのイベントの作成] Web ページに直接移動します。

1. **イベント タイトル:** イベントの名前を入力します。 AltspaceVR の Web サイトと世界のインターフェイスでさまざまなカレンダー ビューを表示する場合は、それを具体的かつ簡潔にしてください。 単語間のスペースを含め、タイトルを 23 文字以下にしてください。
2. **説明:** イベントの説明を入力します。
    * 説明に少なくとも 10 文字が必要です。または、イベントが作成されません。
    * 段落間に空のスペースを追加します。
    * イベントの Facebook、Discord、Web サイト、または他のリソースへのリンクを追加するには、次の形式を使用します (この Markdown は、Web サイトのイベントのプロモーション ページでのみ機能し、AltspaceVR メニューでは正しくレンダリングされません)。 `[Event Name](http://example.com/)`

3. **開始日/終了日:** 開始時刻を設定し、終了時刻が開始時刻より後である必要があります。
4. **カテゴリ:** イベントの種類を最も適切に説明するカテゴリを選択します。 
5. イベントを [プライベート] **または [パブリック]** に **設定します**。
    * 可視性: パブリック イベントは、AltspaceVR Web サイトのイベント[](https://account.altvr.com/events/all)カレンダーの [すべて] タブに表示され、一般公開されています。
    * プライベート イベントは AltspaceVR イベント カレンダーには表示されません。また、イベント空間に入るイベント URL が必要です。
    * "メイン イベントとして追加" する場合は、イベントを public に設定する必要があります。
6. **テンプレートを選択します。** Web ページの右側には、AltspaceVR で使用可能なテンプレートのサムネイル画像の一覧が表示されます。 ゲーム ルーム、オフィス、会議スペース、プレゼンテーション スペース、楽しい会議スペースがあります。 興味深いものを選択します。 気に入らない場合は、新しいテンプレートを使用して新しいイベントを作成してください。 独自のカスタム イベント ワールドを作成する場合は、既定のテンプレートとして **[クール** 空] を選択し、次の手順に従ってワールドをインポートします。
7. [詳細 **オプション] を選択する**

### <a name="advanced-options"></a>[詳細オプション]

1. **昇格:** 各イベントには 2 つのブランドイメージが必要です (これらの画像は AltspaceVR Web サイトに表示され、AltspaceVR 内のイベントには表示されません)。
    * **タイル:** タイル イメージは 1920x1080 ピクセルである必要があります。 この画像は、さまざまなサムネイル サイズにスケーリングされ、他のイベントと一緒に AltspaceVR イベント カレンダーに表示されます。 画像がクリアで、テキストがあまり含めず、その画像に大きなテキストが含めずに表示されます。 所有しない著作権で保護された画像/ロゴは使用しない。
    * **バナー:** バナー 画像は 1920x576 である必要があります。 これは、必要に応じて自動的にサイズ変更され、イベント情報または Web ページに表示されます。 半透明のオーバーレイは、画像の下位 25% をカバーします。 その領域にテキストを配置しないようにします。
    * もう 1 つの (より簡単な) オプションは、[タイル イメージを背景バナー イメージにコピーする] です。これにより、タイル イメージもバナー イメージとして使用されます。 試してみる:)。 見た目が良くなかった場合は、新しいバナー イメージを作成できます。
2. **VR の場合:** このセクションには、AltspaceVR アプリ内のイベント中の仮想エクスペリエンスに適用される機能が含まれています。
    * **既定のコンテキスト ロール:** 緑色の疑問符には、 イベントのすべての対象ユーザー メンバーが使用できる *特定のロールが一覧表示されます*。 最も一般的なは、 *megaphone_only。* これを追加して、[ホスト ツール] ボタンの下にある [自分の声を増幅する] オプションをすべての対象ユーザー メンバーに提供し、大きなスペースの場合はイベント全体で聞こえます。 イベントでフライトが必要な場合は、パイロット を *追加します*。 両方を追加するには、次の形式を使用megaphone_only *します*。 各ユーザーは、[設定]/[入力]/[Fly] に移動して、AltspaceVR アプリで Fly モードを有効にする必要があります。
    * **手順:** 命令 [は、](../world-building/adding-welcome-messages.md) スペースに到着するとウェルカム イメージを生成するテキスト用です。 ユーザーがビューから削除するには、[OK] を選択する必要があります。 これは、多くの場合、"Please stay muted until invited to speak" (ABC をカバーする XYZ イベントへようこそ) などの対象ユーザーの指示を提供するために使用されます。 これは必須ではないので、十分に使用してください。 Markdown を使用して、色とフォントのサイズ設定を追加できます。詳細については、以下をご覧ください。 [http://digitalnativestudios.com/textmeshpro/docs/rich-text](http://digitalnativestudios.com/textmeshpro/docs/rich-text)
3. **詳細:** イベントの高度な機能の簡単な説明を次に示します (これらの機能の一部は、イベントが作成され、イベントを編集するまで表示されません)。
    * **管理者:** 管理者は、イベントの管理に役立つ AltspaceVR ユーザーです。 これらは、コホストまたはバックアップ ホストの場合があります。 [管理者] リストにユーザー名を追加すると、次の機能を実行できます。
        * イベントのタイトル、説明、その他の機能を変更します。
        * モデレーター、ホスト、その他のロールのコンテキスト ロールを追加および削除します。
        * イベントを削除します。
    * **グループ:** ドロップダウン メニューを使用 [して、](group-features.md) プライベート グループから選択します。 アカウントがグループに作成または追加されている場合にのみ表示されます。
    * **Tagline:** この簡潔な文は、タイトルの下のイベント Web ページに表示されます。
    * **[Import from World]/(世界からインポートする):** 作成されたワールド イベント空間を繰り返し使用するために使用する場合、これは、その世界のレイアウトとカスタマイズをイベントにインポートするために使用するフォームのセクションです。 次の考慮事項に注意してください。

    **独自の世界をインポートする:** ワールドをインポートするには、以下を作成して所有しています。
    * ドロップダウン矢印を選択して、作成したワールドの一覧を表示します。
    * ワールドを選択します。
    * イベント フォームの残りの部分を完了した後、少なくとも 2 分待って AltspaceVR のイベント空間にアクセスし、データベース情報が更新され、世界が生成されます。
    
    **他のユーザーの世界を使用する:**
    * 世界の所有者に連絡して、個々の世界で友人と共有機能を使用して共有を依頼してください。

3. **YouTube ビデオ ID:** イベントの Web ページに表示されます。これにより、YouTube イベント トレーラーまたはビデオが、ワールド内ではなく、イベント Web ページに追加されます。 URL のビデオ ID 部分だけが必要です: dQw4w9WgXcQ
4. **Twitter ハンドル:** これにより、Twitter ストリームがイベントの Web ページに追加されます。 Twitter アカウントが個人であり、イベントや関連付けに関連していない場合は、過剰に共有されている可能性があります。 ハンドルだけが必要です。 @elonmusk
5. **コンテキスト ロール:** ここでは、イベント内のイベント ホスト、モ [](../world-building/granting-roles.md)デレーター、その他のロールのスーパーパワーまたはコンテキスト ロールを制御します。 それらを追加するには、AltspaceVR ユーザー名を入力し、ドロップダウン メニューでロールを割り当てる必要があります。 削除するには、[削除] チェック ボックスをオンにして、イベント Web ページを保存します。 注: ホスト モデレーション機能を付与する場合は、ホストとモデレーターとして追加する必要があります。 現在、次のコンテキスト ロールを使用できます。
    * **ホスト:** AltspaceVR [の個々](../faqs/front-row-events.md) のインターフェイスに Front Row 機能を追加して、これらの機能を追加します。
    * **Moderator:** 各参加者にテキストを送信したり、イベントに対してグローバルにミュートしたり、イベントから削除したりする機能など、モデレーション機能を個人のインターフェイスに追加します。 ホストを再度追加し、モデレーション特権を持つ必要がある場合はモデレーション ロールを付与します。
    * **Megaphone のみ:** [ホスト ツール] ボタンを使用して、[自分の声を増幅] ボタンをインターフェイスに追加します。
    * **Terraformer:** [ワールド エディター] ボタンをインターフェイスに追加します。
    * **パイロット:** その個人のフライト機能を追加します。 注: 設定/入力/フライでこれを有効にする必要があります
    * **[楽楽者]:** 音楽イベントと音楽イベントに関連付けられている機能と機能を追加します。
6. **[一覧表示されたユーザーをブロックする]:** ユーザーのアクセスをブロックする場合、またはユーザーがモデレーターまたはホストによってイベントから以前に削除され、イベントが重複している場合は、リストされたユーザーの名前が表示されます。 イベント ホストまたは管理者は、ブロックリストに登録されているユーザーをいつでも追加または削除できます。

フォームが完成し、トリプル チェックが完了したら **、CREATE EVENT を選択します**。

> [!IMPORTANT]
> イベントが正しく作成されていない場合は、説明に少なくとも 10 文字が含まれています。また、テンプレートを選択した場合は、テンプレートの名前が白から青に変わるのを確認してください。

## <a name="event-page-actions"></a>イベント ページのアクション

[イベント] Web ページには、次のアクションとオプションがあります。

1. [ **編集] を** 選択して、イベント設定を変更します。
2. [ **イベントの終了]** を選択してイベントを終了します (たとえば、イベントが早い段階で終了している場合)。
3. [SET **TO DRAFT]を** 選択して、イベントを [アクティブ] ではなく [ドラフト] に設定します。
4. インポートしたワールドを更新した場合は、イベントに表示する変更に対して **[RE-IMPORT WORLD]** を選択する必要があります。イベント の [スペースのリセット] を忘れないでください:)
5. [DUPLICATE **EVENT]を選択** して、今後のイベントのイベントを複製します。 その後、この重複が [マイ イベント]/[マイ ドラフト イベント]に表示されます。新しい日/時刻を更新し、そこからアクティブ化する必要があります。
6. [ADD **AS MAIN EVENT]を選択して** 、カレンダーの一覧にイベントを設定します。
7. イベント **を完全に** 削除する場合は、DELETE EVENT を選択します (これを回復できません)。

準備ができたら、AltspaceVR のワールド内インターフェイスの [イベント **] >** [マイ イベント] メニューに移動し、イベント領域を入力して領域のリセット、変更の検査、またはカスタマイズを続行します。 

> [!IMPORTANT]
> これはコピーであり、イベント空間のカスタマイズは、このイベント専用です。 イベントの領域のカスタマイズに多くの時間を費やしている場合は、繰り返し使用イベント のワールドを作成 **する方が良い場合があります**。

## <a name="using-the-multimedia-console-for-a-slide-presentation"></a>スライド プレゼンテーションにマルチメディア コンソールを使用する

マルチメディア コンソールは、画像、オーディオ、ビデオなどのプレゼンテーション用にコンテンツを整理するための最適な方法です。 マルチメディア コンソール [の指示に従](multimedia-console.md) って、イベントのセットアップを取得します。

## <a name="branding-your-event-with-images"></a>画像を使用したイベントのブランド化

AltspaceVR では、イベント空間や会社のロゴやブランド、自分やチームなどのビジュアル コンテンツの写真に画像を簡単に追加できます。

イメージの要件は次のとおりです。
* JPG または PNG ファイルを受け入れる
* イメージの最大サイズ: 1920 x 1080
* 解決策:72 DPI
* ファイルサイズ: ファイルあたり 250 KB 未満

このプロセスでは、イメージを [altspace Evr の写真](https://account.altvr.com/photos)にアップロードし、世界中のエディターの写真を使用して、イベントのワールド空間にインポートして配置します。

1. Altworkplace Evr web サイトの [写真](https://account.altvr.com/photos) にアクセスします。
2. **[アップロード]** を選択します。
3. [ **参照** ] を選択して、コンピューター上でダイアログボックスを開き、最適化されたイメージを探して選択します。
4. **[Open]** を選択します。
5. [ **写真の作成**] を選択します。
6. すべてのイメージに対してこの手順を繰り返します。

Altplace Evr およびイベントの世界では、イメージを配置する場所の近くに配置します。 (これを **繰り返し使用するイベント** 用に作成している場合は、イベント領域ではなく、世界中のユーザーであることを確認してください)。

1. ユーザーインターフェイスの右下隅にある [**ワールドエディター]/[エディターパネル**] を選択します。
2. 下部にある [ **自分** のもの] タブをクリックします。
3. **写真** を選択します。
4. 写真はアップロードの時間順に保存されるため、アップロードされた最新のイメージは一覧の最初のイメージになります。 これを選択して、世界に追加します。
5. デバイスカーソルを使用して、イメージを取得し、目的の場所に配置します。
    * コントローラーを使用してサイズを変更することができます (通常は、VR のタッチパッド全体で thumb を水平にスライドさせるか、2D モードでマウスを使用します)。
    * 画像の位置と回転を微調整するには、エディターでイメージの **歯車** 記号を選択し、回転と位置を適切に設定します。
    * また、スケール機能を使用してイメージのサイズを変更することもできます。

> [!NOTE]
> 編集モードがオン (オレンジ色のテキスト) になると、自動的にフライトが可能になり、イベントの世界でのイメージの配置がより簡単になります。

6. イメージの配置ごとにこの手順を繰り返します。
7. イメージが適切に設定されている場合は、個別にロックしていない場合は [ **すべてロック** ] を選択し、編集モードをオフにして、エディターを閉じます。
8. イベントあたり最大で5枚の写真を使用します。

## <a name="adding-3d-objects-using-world-editor-and-kits"></a>ワールドエディターとキットを使用した3D オブジェクトの追加

Altworkplace Evr を使用すると、ワールドエディターで3D オブジェクトをワールドにインポートできます。 現在、使用可能なさまざまなエディターキットが用意されていますが、その方法もあります。 宇宙船、rockets、アニメーション化された動物、sculptures、木、植物、およびさまざまなオブジェクトがあります。 または、独自のキットをパブリックコレクションに追加することも、個人の使用を目的として使用することもできます。または、対話型オブジェクトをヘルメット、swords、主翼、衣料、帽子などの対話型オブジェクトとして使用する [ことを検討](../world-building/using-mixed-reality-extensions.md)してください。 オブジェクトの一覧が間もなく拡張されることが予想されます。

ここでは、既にキットに含まれている項目の使用について説明します。 3D オブジェクトを Altに追加してインポートする方法の詳細については、「 [glTF モデルを自分の世界にインポートする方法」](../world-building/importing-models.md)を参照してください。 また、 [「独自のキットをアップロード操作方法ですか](../world-building/uploading-custom-kits.md)。

世界中のビルドキットからイベント世界に既存の3D オブジェクトを追加するには、[ワールドエディター]/[エディター] パネルを開いて編集モードを有効にします。

1. インポートするオブジェクトのキットを探索して選択します。
2. カーソルを使用してキットオブジェクトを選択し、その位置に位置付けます。 ワールドエディターのオブジェクトの歯車アイコンを使用して、手動の番号入力の位置、回転、およびスケールのオプションを使用します。
3. オブジェクトが右側の位置に配置されたら、ロックアイコンをクリックしてロックします。
4. 必要に応じて、別のオブジェクトを追加します。 3D オブジェクトを追加する際には慎重に行う必要があります。5つの異なる個別のキットの項目を選択したままにしておくと、Oculus クエストユーザーサポートが向上します。 世界では、200を超えるオブジェクトを使用しないでください。
5. 完了したら、すべての項目をロックし、編集モードをオフにして、ワールドエディターを閉じます。 移動します。 いくつかのテスト担当者を招待して、アクセスして確認し、優れたイベントの準備をしましょう。

ワールドエディターキットの詳細と、次のように3D オブジェクトを Altworkplace Evr に追加する方法については、以下を参照してください。

* [早期アクセスプログラムとは何ですか。](../world-building/early-access.md)
* [GlTF モデルをワールドにインポートする方法](../world-building/importing-models.md)
