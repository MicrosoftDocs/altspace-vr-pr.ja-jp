## <a name="how-to-use-patreon-and-eventbrite-to-host-a-ticketed-vr-event"></a>チケット VR イベントをホストするために、測定済みの Reon と Eventbrite を使用する方法は次のとおりです。

### <a name="creating-a-list"></a>リストを作成する

スプレッドシートプログラムを使用して、すべての電子メールアドレスを含む単一の列を作成します。 これが完了したら、次の手順を実行します。  

1. グループを作成または編集します。  
2. [ **メンバーの電子メール** ] フィールドを検索します。  
3. * .csv ファイルから電子メールをインポートするには、 **[ファイルの選択** ] オプションを探します。
4. リストは、1行に1つの電子メールでインポートする必要があります。

> [!NOTE]
> このオプションを選択した場合は、* .csv ファイルからインポートする代わりに、電子メールをコピーしてフィールドに貼り付けることもできます。

### <a name="importing-a-list"></a>リストのインポート

測定値の再設定や Eventbrite などのプラットフォームでは、専用の "電子メール" 列を含む * .csv ファイル (コンマ区切り値) をエクスポートできます。この列は、使用するために選択できます。これを利用できるようになったら、ほとんどの手順は同じです。  

1. グループを作成または編集します。  
2. [ **メンバーの電子メール** ] フィールドを検索します。  
3. * .csv ファイルから電子メールをインポートするには、 **[ファイルの選択** ] オプションを探します。
4. リストは、1行に1つの電子メールでインポートする必要があります。

> [!NOTE]
> このオプションを選択した場合は、* .csv ファイルからインポートする代わりに、電子メールをコピーしてフィールドに貼り付けることもできます。

**測定される再オン ** 

[測定](https://www.patreon.com/)される再オン  は、クリエイティブがコンテンツのサブスクリプションプログラムを作成できるメンバーシッププラットフォームです。Pledge のデータをエクスポートする方法は複数ありますが、ここでは CSV の方向に注目します。 この情報については、 [こちら](https://www.patreon.com/portal/how-to/export-pledge-data)に記載されているヘルプページを参照する   か、以下の手順に従ってください。  

1. "得意客" フィールドは、[作成者] サイトの左側にある [creator] サイドバーで検索します。  
2. これを展開して "リレーションシップマネージャー" を検索し、それを選択します。  
3. ページの上部には、CSV というダウンロードボタンがあります。 それを選択します。  
4. ファイルを保存するための windows ダイアログフィールドがポップアップ表示されます。ファイルを保存します。  
5. このファイルを使用するには、[Altworkplace Evr] グループページのインポート手順に従います。  

**Eventbrite ** 

[Eventbrite](https://www.eventbrite.com)  は、ユーザーがイベントのチケットを管理および作成できるようにするチケット web サイトです。測定の再有効化と同様に、出席者の電子メールリストを生成する方法は多数あります。CSV の方向性についても説明します。 この情報のヘルプページは [こちらで](https://www.eventbrite.com/support/articles/en_US/How_To/how-to-edit-and-export-an-attendee-summary-report?lg=en_US)見つけることができ   ます。また、このデータを抽出する方法の1つとして、以下の手順に従うこともできます。  

1. プロファイルメニューから、[イベントの管理] メニューを探します。 
2. 出席者の電子メールの一覧を取得するイベントを選択します。 
3. イベントのダッシュボードで、"Common Reports" というカテゴリを見つけます。 
4. また、左側の列メニューの [Analyze-> イベントレポート] の下にある [共通レポート] を見つけることもできます。   
5. [参加者の概要] を選択します。 
6. "レポートの結果" セクションを検索します。 
7. [CSV] ボタンを選択して、ファイルをダウンロードします。
