## <a name="how-to-add-an-email-list-to-a-group"></a>グループに電子メール リストを追加する方法

### <a name="creating-a-list"></a>リストの作成

スプレッドシート プログラムを使用して、すべての電子メール アドレスを含む 1 つの列を作成します。 これを完了したら、次の操作を行います。  

1. グループを作成または編集します。  
2. [メンバー **の電子メール] フィールドを見** つける。  
3. [ファイルの **選択]** オプションを見つけて、*.csv ファイルから電子メールをインポートします。
4. リストは、1 行につき 1 つの電子メールでインポートする必要があります。

> [!NOTE]
> 選択した場合は、*.csv ファイルからインポートするのではなく、フィールドに電子メールをコピーして貼り付けすることもできます。

3. [ファイルの **選択]** オプションを見つけて、*.csv ファイルから電子メールをインポートします。
4. リストは、1 行につき 1 つの電子メールでインポートする必要があります。
