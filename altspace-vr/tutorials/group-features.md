### <a name="creating-a-list"></a>リストを作成する

スプレッドシートプログラムを使用して、すべての電子メールアドレスを含む単一の列を作成します。 これが完了したら、次の手順を実行します。

3. * .csv ファイルから電子メールをインポートするには、 **[ファイルの選択** ] オプションを探します。
4. リストは、1行に1つの電子メールでインポートする必要があります。

> このオプションを選択した場合は、* .csv ファイルからインポートする代わりに、電子メールをコピーしてフィールドに貼り付けることもできます。

### <a name="importing-a-list"></a>リストのインポート

測定値の再設定や Eventbrite などのプラットフォームでは、専用の "電子メール" 列を含む * .csv ファイル (コンマ区切り値) をエクスポートできます。この列は、使用するために選択できます。これを利用できるようになったら、ほとんどの手順は同じです。  

1. グループを作成または編集します。  
2. [ **メンバーの電子メール** ] フィールドを検索します。  
3. * .csv ファイルから電子メールをインポートするには、 **[ファイルの選択** ] オプションを探します。
4. リストは、1行に1つの電子メールでインポートする必要があります。

> [!NOTE]

### <a name="i-want-to-use-eventbrite-and-patreon-but-i-dont-see-where-to-get-the-email-addresses-for-my-groups"></a>Eventbrite と Patreon を使用したいが、グループのメール アドレスを取得する場所が表示しない

**Patreon ** 

[Patreon](https://www.patreon.com/)  は、クリエイティブがコンテンツのサブスクリプション プログラムを作成できるメンバーシップ プラットフォームです。Patreon には複数の方法でテキスト データをエクスポートする方法はありますが、CSV の方向に焦点を当てる必要があります。 この情報のヘルプ ページについては、こちらを [参照](https://www.patreon.com/portal/how-to/export-pledge-data)するか   、以下の指示に従ってください。  

1. Patreon サイトの左側にある作成者のサイド バーにある [お願い] フィールドを探します。  
2. それを展開して "リレーションシップ マネージャー" を見つけて選択します。  
3. ページの上部には、CSV というダウンロード ボタンがあります。 それを選択します。  
4. ファイルを保存するためにウィンドウ ダイアログ フィールドがポップアップ表示されます。ファイルを保存します。  
5. AltspaceVR グループ ページのインポート手順に従って、このファイルを使用します。  

**Eventbrite ** 

[Eventbrite](https://www.eventbrite.com)  は、ユーザーがイベントのチケットを管理および作成できるチケット Web サイトです。Patreon と同様に、出席者の電子メール リストを生成する方法は多数あります。もう一度 CSV の方向に注目します。 この情報のヘルプ ページはこちらで確認 [](https://www.eventbrite.com/support/articles/en_US/How_To/how-to-edit-and-export-an-attendee-summary-report?lg=en_US)できます。または、このデータを抽出する方法の 1 つについて、以下の   指示に従ってください。  

1. [プロファイル] メニューの [イベントの管理] メニューを探します。 
2. 参加者の電子メール リストを取得するイベントを選択します。 
3. イベントのダッシュボードで、"共通レポート" というカテゴリを見つける。 
4. 左側の列メニューの [分析 ] -> イベント レポート] にも [共通レポート] が表示されます。   
5. [参加者の概要] を選択します。 
6. [レポート結果] セクションを見つける。 
7. [CSV] ボタンを選択してファイルをダウンロードします。 

> [!IMPORTANT]
> これはベータ版の機能です。 AltspaceVR は無料のエクスペリエンスであり、参加者がチケット付きイベントにアクセスできない可能性があるエクスペリエンスで発生する可能性のある技術的な問題に対して責任を負いません。
