## <a name="recording-altspacevr-in-2d-mode-on-pc"></a>PC の2D モードでの Altを記録する

### <a name="the-short-version"></a>短いバージョン

1. Altの Evr と OBS がインストールされていること。 [Alt] を2D モードで起動し、OBS を起動します。また、[Altを記録する] に OBS を設定して、記録します。

### <a name="the-slightly-longer-version"></a>少し長いバージョン

1. [https://obsproject.com/](https://obsproject.com/) にアクセスします
2. [ **Windows** ] を選択して、OBS をダウンロードします。 この投稿は **OBS v 22.0.2** を使用しています
3. OBS をインストールする

### <a name="have-altspacevr-running-in-2d-mode-before-you-run-obs"></a>OBS を実行する前に、2D モードで実行しています。

1. Web サイトから[altaltvr.com/get](https://altvr.com/getaltspacevr)をダウンロードしてインストールします。
2. PC から HMD の USB ケーブルを取り外すことによって、または Rift: Ctrl + Alt + Del、Services、Oculus VR Runtime Service を持っている場合は、右クリックして、[停止] をクリックして、必ず Altを起動してください。 
    * これにより、Oculus が無効になり、2D モードで Alt Evr が開始されます。これらの手順を繰り返し、Start を使用して VR モードを戻します。

次に、OBS に Alt-Tab ます。

1. [ソース] で、[ **+ > ゲームキャプチャ > 新規作成**] を選択します。
2. テキストを ' Altworkplace Evr Capture ' に編集し、tick **Make ソースを表示** して、[OK] を選択します。
3. [ソース] の [ **alt]** をダブルクリックします。
4. **特定のウィンドウをキャプチャ** する **モード** の変更
5. ウィンドウ: [AltspaceVR.exe]: Altworkplace Evr
6. ウィンドウの一致の優先度: タイトルと一致します。それ以外の場合は、同じ実行可能ファイルのウィンドウを検索します。
7. キャプチャカーソルまで下へスクロール: untick
8. [OK] を選択します。

これにより、「OBS」に Altが表示されるようになります。 OBS で次のプロパティを設定するには、[**ファイル > 設定** にアクセスします。

2. OBS に Alt-Tab し、準備ができたら [ **記録の開始**] をクリックします。

OBS の右下にあります。これは、カウントが開始され、ドットが赤であることを示しています。

次のようなテストの記録を作成します。 
1. Altworkplace Evr のメニューを開いたり、閉じたり、ロールオーバーしたりして、UI サウンドを作成します
2. Unmuted であることを確認してください。または、通常のボリュームで別のユーザーと話し合ってください。
3. 動作しているかどうかを確認するには、デスクトップの Audio および Mic/Aux レベルを確認します。

通常、録音時に Mic/補助をミュートします。 マイク/補助のスピーカーアイコンを選択すると、赤が X に変わります。

* オーディオとその他のユーザーのオーディオを一致させることが非常に困難であるため、イベントを記録しているときは Mic が最適にミュートされます。
* 音声に関するもう1つの問題は、OBS の設定方法です。 コンピューターからすべてのオーディオをキャプチャするので、PC で YouTube を視聴している場合は、オーディオまたは Discord の通知が記録されます。
* [alt] からのオーディオのみを記録するには、ボリューム Mixer (Windows の右下にあるスピーカーアイコンを右クリック) に切り替え、システムサウンドやブラウザーなどをミュートにします。ただし、OBS または altにはミュートを入れないでください。

> [!IMPORTANT]
> 記録後に、これらのボリューム Mixer 設定を再び有効にするようにしてください。

次に、OBS に戻り、[ファイルからの **記録の停止** ] を選択して>記録を **表示** します。 これにより、OBS ビデオファイルを含むフォルダーが開き、テストビデオがダブルクリックされます。

録音が非常に大きい場合もあるため、デスクトップオーディオのスライダーを下げて、別の記録をテストすることができます。


## <a name="live-streaming-to-youtube-in-altspacevr-2d-mode-on-pc"></a>PC の Altworkplace Evr 2D モードで YouTube へのライブストリーミングを行う

### <a name="the-short-version"></a>短いバージョン

Altの Evr と OBS がインストールされていること。 [Alt] を2D モードで起動し、OBS を起動します。または、YouTube で "新しいライブイベント" を作成するか、youtube ストリームキーを使用して OBS を設定し、[OBS でストリーミングを開始します]、[youtube でストリーミングを開始します] をオンにします。

### <a name="the-slightly-longer-version"></a>少し長いバージョン

1. [https://obsproject.com/](https://obsproject.com/) にアクセスします
2. **Windows** を選択して obs をダウンロードします (この投稿は obs v 22.0.2 を使用しています)
3. OBS をインストールする

OBS を実行する前に AltspaceVR を 2D モードで実行する
1. Web サイトから AltspaceVR をダウンロードします。 [https://account.altvr.com/downloads](https://account.altvr.com/downloads)
2. 2D モードで AltspaceVR を起動するには、HMD の USB ケーブルを PC から取り外すか、リフトがある場合は Ctrl + Alt + Del、Services、Oculus VR Runtime Service を右クリックし、[停止] をクリックします。 これにより、Oculus Home が無効にされ、2D モードで AltspaceVR が起動されます。これらの手順を繰り返し、[開始] を選択して VR モードを再度取得します。

3. Alt-Tab OBS に渡す

4. [ソース] で、 [ゲーム キャプチャの選択] 、 [新規作成] 、 [AltspaceVR Capture] へのテキストの編集] を選択し、[ ソースを表示する] をオンにし、[ **+** OK] を選択します。
5. AltspaceVR Capture をダブルクリックします
6. モード: 特定のウィンドウをキャプチャする
7. ウィンドウ: [AltspaceVR.exe]: AltspaceVR
8. ウィンドウの一致の優先度: タイトルと一致します。それ以外の場合は、同じ実行可能ファイルのウィンドウを検索します
9. [Capture Cursor]/(カーソルのキャプチャ) まで下へスクロールします。 [OK] を選択解除します

これにより、AltspaceVR が OBS に表示されます。 成功です。

| オーディオ | オーディオ ビットレート: 160 (すべてのトラックの場合) サンプル レート: 48khz <br>チャネル: ステレオ <br>デスクトップ オーディオ デバイス: 既定 <br>デスクトップ オーディオ デバイス 2: 無効にする <br>Mic/Aux Audio Device: Default |

3. ページの左側のライブ ストリーミング。

'**Stream now**' method:

* [編集] を選択してライブ ストリーム情報を入力します<br>
