## <a name="recording-altspacevr-in-2d-mode-on-pc"></a>PC での 2D モードでの AltspaceVR の記録

### <a name="the-short-version"></a>短いバージョン

1. AltspaceVR と OBS (Open Broadcaster Software) をインストールします。 2D モードで AltspaceVR を起動し、OBS を起動し、OBS を設定して AltspaceVR を記録して記録します。

### <a name="the-slightly-longer-version"></a>少し長いバージョン

1. [https://obsproject.com/](https://obsproject.com/) にアクセスします
2. [WINDOWS]**を** 選択して OBS をダウンロードします。 この投稿では **OBS v26.1.1 を使用しています**
3. OBS をインストールする

### <a name="have-altspacevr-running-before-you-run-obs"></a>OBS を実行する前に AltspaceVR を実行する

1. Web サイトから AltspaceVR をダウンロードしてインストールします: [altvr.com/get](https://altvr.com/getaltspacevr)
2. VR ビデオを安定させて頭の動きを解消する場合は、2D クライアントを使用するか、ヘッドセットの USB ケーブルを PC から取り外して 2D モードで AltspaceVR を起動してください。 リフトがある場合は、Ctrl + Alt + Del キーを押し、[ **サービス**] 、 **[Oculus VR Runtime Service]** を選択し、右クリックして [停止] を選択 **します**。 これにより Oculus が無効にされ、2D モードで AltspaceVR が起動されます。 これらの手順を繰り返し、[開始] を使用して VR モードを戻します。
3. OBS でのゲーム キャプチャを使用して、VR モードでエクスペリエンスを記録することもできます

次に、Alt-Tab OBS に渡します。

1. [ **シーン] で**、新 **+** しいシーンを選択して名前を指定します。
2. 次に、[ソース **] で**、[+ Game Capture >作成] **>選択します。**
2. [AltspaceVR Capture] にテキストを編集し、[ソースを表示する] をオン **に** し **、[OK] を選択します。**
3. [ソース] の **下にある AltspaceVR Capture** **をダブルクリックします。**
4. [モード **] を [****特定のウィンドウをキャプチャする] に変更する**
5. ウィンドウ: [AltspaceVR.exe]: AltspaceVR
6. ウィンドウの一致の優先度: タイトルと一致します。それ以外の場合は、同じ実行可能ファイルのウィンドウを検索します
7. 下へスクロールして **Capture Cursor**: untick
8. **[OK]** を選択します。
9. これにより、AltspaceVR が OBS に表示されます。

OBS で次のプロパティを設定するには、 [ファイル] に移動 **> 設定。**

1. Alt-Tab AltspaceVR に移動し、正しい空間/ワールド/イベントに入り、ビデオを記録する予定のカメラ (つまり、アバター) を表示します。
2. Alt-Tab OBS にアクセスし、準備ができたら [記録の開始] **をクリックします**。

OBS の右下に **、REC:** がカウントを開始し、ドットが赤で表示されます。これは、記録中です。

このテスト記録を作成します。 
1. AltspaceVR でメニューを開く/閉じる/ロールオーバーして UI をサウンドにする
2. "Sibilance, Sibilance" など、変更されていないか、別のユーザーに通常のボリュームで話しかけさせる必要があります。
3. デスクトップ オーディオと **マイク/Aux** のレベルを確認して、オーディオが適切に選択されていないか確認します。

通常、録音時に Mic/Aux をミュートします。 先に進み、Mic/Aux のスピーカー アイコンを選択すると、X で赤に変わります。

* マイクのオーディオ レベルを他のユーザーのオーディオと一致するのは非常に難しいので、イベントを記録するときにマイクが最適にミュートされます。
* オーディオに関するもう 1 つの問題は、OBS の設定方法です。 お使いのコンピューターからすべてのオーディオがキャプチャされます。そのため、YouTube を視聴している場合、または PC で通知サウンドを受け取る場合は、そのオーディオが記録されます。
* AltspaceVR からのオーディオを録音するには、Open **Volume** ミキサーに移動し (Windows の右下にあるスピーカー アイコンを右クリック)、システム サウンドやブラウザーをミュートしますが、OBS や AltspaceVR はミュートされません。

> [重要]記録後は、これらのボリューム ミキサー設定のミュートを忘れずに解除してください。

ここで、OBS に戻り、[ 記録の停止] **を選択します**。 記録したビデオを見つけるには、記録の表示 **に関する>に移動します**。 これにより、OBS ビデオ ファイルを含むフォルダーが開き、テスト ビデオをダブルクリックします。

録音が非常に大きいので、OBS のデスクトップ オーディオのスライダーを下げ、テストする別の録音を行う場合があります。

Proヒント: Ctrl + Alt + P キーを使用して Toggle をオンにし、ビューから UI を削除して、クリーン ショットを取得します。

AltspaceVR ビデオ レコーダーです。

## <a name="live-streaming-to-youtube-in-altspacevr-2d-mode-on-pc"></a>PC 上の AltspaceVR 2D モードで YouTube にライブ ストリーミングする

### <a name="the-short-version"></a>短いバージョン

Altの Evr と OBS がインストールされていること。 Altworkplace Evr と OBS を起動します。 ライブストリーム ' Now ' または ' 後の日付 ' を指定できます。 YouTube で、YouTube ストリームキーを使用して OBS を設定します。 OBS と YouTube でのストリーミングを開始し、レースを終了します。

## <a name="setting-up-live-streaming-on-youtube"></a>YouTube でのライブストリーミングの設定

ライブストリームを取得するには、"すぐに" 使用するか、未来のライブストリームを "後の日付" で設定します。

1. ブラウザーを開き、サインインして、 [https://www.youtube.com/](https://www.youtube.com/) にアクセスします。 [https://studio.youtube.com/](https://studio.youtube.com/)
2. 右上にある [**作成**] を選択し、[**ライブ**] を選択します。

**' Right now '** メソッド:

1. 今すぐ選択する **/開始**
1. **ストリーミングソフトウェア/外出先** の選択
1. ビデオの詳細とカスタマイズを編集するには、[ **編集**]、[右上] を選択します。
1. [ **Stream 設定** で、既定値のままにします。
1. [ **Stream key (エンコーダーに貼り付ける)**] の横にあるキーを **コピー** して、OBS に貼り付けます。
1. OBS/**設定**  /  **ストリーム** を開く
1. [**サービス**] ドロップダウンメニューで、[ **YouTube-rtmps** ] を選択します。
1. YouTube のストリームキーを OBS の **ストリームキー** フィールドに貼り付けます
1. [**適用**] をクリックし、[ **OK]** をクリックします。
1. OBS で [ **ストリーミングの開始** ] を選択する
1. YouTube に切り替えると、YouTube に住んでいることがわかります。
1. 実際の YouTube ライブストリームビデオページを表示するには、右上にある [共有] アイコンを選択する必要があります。
1. [ビデオリンク] をクリックすると、YouTube ライブストリームが表示され、再生されます。 
1. この URL はライブストリームリンクであり、すべてのソーシャルチャネルに対して共有でき:)
1. ライブストリームを停止するには、[YouTube でストリームを終了] を選択し、OBS でストリーミングを停止します。

'**後の日付**' メソッド:
1. 左上にある [ **管理** ] アイコンを選択します。
1. **スケジュールストリーム**(右上) を選択します
1. タイトル、説明、カテゴリ、サムネイル (1280 0x720) を追加してから **次へ**
1. ライブチャットオプション、[**次へ**]
1. 非公開、一覧にない、またはパブリック (パブリックを選択)
1. 有効にする日付と時刻をスケジュールし、**完了** します

 **今後、ライブストリームを開始する準備ができたら、次のようになります。**
1. [Stream 設定で、既定値のままにします。
1. [ **Stream key (エンコーダーに貼り付ける)**] の横にあるキーを **コピー** して、OBS に貼り付けます。
1. OBS/**設定**  /  **ストリーム** を開く
* [**サービス**] ドロップダウンメニューで、[ **YouTube-rtmps** ] を選択します。
1. YouTube のストリームキーを OBS の **ストリームキー** フィールドに貼り付けます
1. [**適用**] をクリックし、[ **OK]** をクリックします。
1. OBS で [ **ストリーミングの開始** ] を選択する
1. YouTube に切り替えると、YouTube に住んでいることがわかります。
1. 実際の YouTube ライブストリームビデオページを表示するには、右上にある [共有] アイコンを選択する必要があります。
1. [ビデオリンク] をクリックすると、YouTube ライブストリームが表示され、再生されます。 
1. この URL はライブストリームリンクであり、すべてのソーシャルチャネルに対して共有でき:)
1. ライブストリームを停止するには、[YouTube で **ストリームを終了** ] を選択し、OBS で **ストリーミングを停止** します。
