# <a name="is-there-a-push-to-talk-button"></a>[プッシュツートーク] ボタンはありますか。

表示の右下には、音声を他のユーザーに共有しているかどうかを切り替えるためのマイクボタンがあります。 または、キーボードショートカットの **Ctrl + Alt + T** を使用して、この設定を切り替えることもできます。 
 
話すときにアイコンが点滅する場合は、マイクが動作しています。
